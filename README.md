# 🔐 Base64 Tool

Encode and decode Base64 strings instantly. Supports text and file encoding — runs 100% in your browser.

## 🚀 Features

- **Text Encode/Decode** — Convert text to/from Base64
- **File Encode** — Upload files to get Base64 string
- **Image Preview** — See decoded images directly
- **URL-Safe Mode** — Support for URL-safe Base64
- **Copy to Clipboard** — One-click copy
- **No Server** — Runs entirely in browser

## 🎯 Why This?

Most Base64 tools are online services. This runs locally — your data never leaves your machine.

## 🛠 Tech Stack

- Next.js 14
- React
- Tailwind CSS
- shadcn/ui

## 📦 Setup

```bash
git clone https://github.com/thehaardik/base64-tool.git
cd base64-tool
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
